HOW TO USE B·FLIX
A
LOAD: Drop a video or photo onto the canvas. It extracts up to 300 frames. Draw on top or run a script over it.
B
CODE: The editor compiles live as you type. Try the preset buttons (RAIN, PULSE…) or hit ASK CLAUDE and describe what you want — Claude writes the code for you.
C
DRAW: Paint directly on the active frame. Pen, Box, Circle, Move, Erase. Colors are Bauhaus-standard. The MOVE tool slides your drawing around the canvas.
D
SAVE: Export as PNG, WebM, MP4, GIF, or a contact sheet showing all frames.
LIVE COMPILER: The B tab compiles as you type — the timeline updates instantly. The cursor position syncs to the frame that code refers to.
→
NAVIGATE: ← → arrow keys, swipe left/right on canvas, or tap frames in the timeline strip.
