ICARO-FLOW V2 · FIRST RUN

ICARO-FLOW V2 is a local spatiotemporal sketchbook for short animated sequences: draw frames, import media, stamp text, run BEFLIX-like commands, inspect motion, then export.

Start with A ADD for blank/media frames, C COLOR for drawing, B BUILD for text/scripts/procedural motion, and D SAVE for PNG/GIF/WebM/project files.

All processing is local unless you export/share the generated files. Analysis overlays are artist-facing diagnostics, not scientific optical flow or physics validation.
