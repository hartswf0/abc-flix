# TAX-FUTURE

This directory contains the development phases of the **Taxonomy of Futures Engine**.

## Contents
Based on the Stuart Candy / Dunne & Raby future cones model (Possible, Plausible, Probable, Preferable), these prototypes iteratively unroll static taxonomy diagrams into a dynamic, operative synthesis phase-space. The collection includes isolated components, morph experiments, and completed synthesis architectures.

For a complete visual overview of all taxonomy diagrams functioning simultaneously, open `index.html`.
