What Is This?

AETHEL-ORCHESTRATOR is the automation engine behind ABC FLIX WorldText generation. It manages a queue of generation tasks, dispatches them into available Sora/Veo/Minimax slots, and tethers the resulting output files back to their source tasks.

Panels

  • S — State: Machine state, slot availability, uptime counter
  • Q — Queue: Full manifest of pending, active, and completed tasks
  • L — Log: Live terminal output from the orchestrator with filter controls
  • ? — Help: This panel

States

  • BOOT — System initializing
  • IDLE — Waiting for tasks
  • POLLING — Checking slot availability
  • SLOT_FOUND — Available slot located
  • UPLOADING — Pushing assets to generation service
  • INJECTING — Writing prompt into slot
  • AWAITING — Generation in progress remotely
  • TETHERING — Binding output file to task
  • ERROR / HALTED — Fault condition

Demo Mode

When the orchestrator is unreachable (ws://localhost:8765), the dashboard runs simulated data so you can explore the interface. A DEMO MODE badge appears in the top-right corner. Connect the real orchestrator with:

node orchestrator.js

Terminal Filters

Use the filter buttons above the log to isolate by level:

  • ALL Show everything
  • STATE Machine state transitions
  • DRIP Queue / scheduler messages
  • HCI Upload and prompt injection events
  • TETHER File binding events
