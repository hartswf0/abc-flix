Architecture: Single-Agent Centralized RL ONE global Q-table: Q(state, action). State = cell position on the 128x96 grid. Actions = {up, right, down, left, stay}. All particles share the same learned policy. Reward = target(next) * 1.15 + climb_bonus * (target(next) - target(cur)) - crowd_penalty * occupancy(next) - 0.008 step cost. The value function V(s) = max_a Q(s,a) shows where the policy has learned to direct flow. Advantage A(s,a) = Q(s,a) - V(s) shows which actions are better than average. Epsilon decays over training to shift from exploration to exploitation. Compare with the multi-agent version to see centralized vs decentralized swarm learning tradeoffs.
