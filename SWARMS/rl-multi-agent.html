Architecture: Multi-Agent RL Each macroblock patch owns an independent Q-table. State = local (x,y) within patch. Actions = {up, right, down, left, stay}. Reward = target density at next cell + gradient climb bonus - crowding penalty - step cost. Patches share boundary gradients with neighbors to encourage cross-patch coherence without a central controller. This is decentralized swarm learning.
