The Rule
A frame is not an image. It is a compressed state of forces. This mode compares one slice to the next, guesses the missing law, marks the wound, and lets a swarm correct only the failed patch.
The Seven Verbs
LOAD imports matter. WOUND compares current to next. LISTEN infers flow and patches. DREAM predicts the next slice. SWARM corrects failed patches. BURN commits belief into matter. FOSSILIZE exports the autopsy.
Best Loop
LOAD → WOUND → LISTEN → DREAM → SWARM → BURN → FOSSILIZE. Do not train the whole frame first. Select or use the worst failed patch.
